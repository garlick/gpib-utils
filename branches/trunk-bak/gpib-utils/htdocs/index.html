gpib-utils

Welcome to gpib-utils. This package contains linux command line utilities for several GPIB-based electronic test instruments.

The hp1630 utility offers configuration save/restore, screen printing, and off-analyzer trace manipulation for the HP 1630/1631 logic analyzer/oscilloscope. Here is a 1631A screen dump of an FM signal source produced with the command: 

hp1630 --screen-dump | hptopbm > file

Here is a screen dump of dinotrace browsing an 1631A timing trace saved in DECSIM ASCI format with the command: 

hp1630 --save-timing | hptodino > file

The hp3455, hp3457, and r5005 utilities can set configurations for HP 3455, HP 3457, and Racal-Dana 5005 digital multimeters. These utilities can also take samples from the instruments at regular intervals.

The hp8656 utility can configure the HP 8656A synthesized signal generator. Here is a frequency response plot of a high pass RC filter taken with the following shell script: 

#!/bin/bash

hp3457 --clear --function acv --trigger syn
hp8656 --clear --frequency 100khz --incrfreq 10khz --amplitude 1v

# Take measurements from 100khz to 1mhz in 10khz increments
for freq in `seq 100 10 1000`; do
  volts=`hp3457 --samples 1`
  echo "$freq     $volts"
  hp8656 --frequency up
done
			
hp3457 --local
hp8656 --local

See the INSTALL file for instructions on setting up with linux-gpib.

For operation information, see the man pages: hp3455(1), hp3457(1), hp8656(1), r5005(1), hp1630(1), hptopbm(1).

The examples directory that is part of the release tarball may also be of use.

For downloads, trackers, etc, please see the project's SF summary page.
