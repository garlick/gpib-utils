HP 1630/1631 Support in gpib-utils

The gpib-utils hp1630 utility offers configuration save/restore, screen printing, and off-analyzer trace manipulation for the HP1630/1631 logic analyzer/oscilloscope.

Here is a 1631A screen dump of an FM signal source produced with the command: 

hp1630 --print-screen | hptopbm | pnmtopng > screen.png

Here is a screen dump of dinotrace browsing an 1631A timing trace saved in DECSIM ASCI format with the command: 

hp1630 --save-timing | hptodino > file
