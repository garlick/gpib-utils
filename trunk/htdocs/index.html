gpib-utils

Simple UNIX/Linux command line utilities for the test bench
gpib-utils supports a variety of instruments and is easily extended to accomodate more (submissions welcome!)

gpib-utils can use linux-gpib software to communicate with instruments using a GPIB USB or PCI adapter on Linux.

Or gpib-utils can also use built-in VXI-11 support to communicate with instruments via Ethernet on any UNIX/Linux system.

Save and restore a test setup. Capture a screen dump. Extract data for gnuplot.

Utilities can be combined in shell scripts to drive light automated test applications.

Download the latest gpib-utils and enjoy!
